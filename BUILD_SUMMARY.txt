ВАЖНО: Сборку нужно выполнять на Windows машине!

Шаг 1: Подготовка
  1. Скопируйте весь проект на Windows компьютер
  2. Установите Python 3.8+ (с python.org)
  3. Откройте Command Prompt (cmd) в папке проекта

Шаг 2: Установка зависимостей
  > python -m venv venv
  > venv\Scripts\activate
  > pip install -r requirements.txt
  > pip install pyinstaller

Шаг 3: Сборка приложения
  > build_windows.bat

  Результат: dist\ImageDataAnnotator\ImageDataAnnotator.exe
  Размер: ~100 МБ

Шаг 4: Создание установщика (опционально)
  1. Установите Inno Setup 6:
     https://jrsoftware.org/isdl.php
  
  2. Запустите:
     > build_installer.bat

  Результат: dist\installer\ImageDataAnnotator-Setup-v1.0.0.exe
  Размер: ~50-70 МБ

⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════════

1. Сборка на macOS не создаст Windows .exe!
   Нужна Windows машина для сборки Windows версии

2. Первый запуск может быть медленным
   Windows Defender может проверять файл

3. Антивирусы могут ругаться
   Это нормально для PyInstaller приложений
   Добавьте в исключения, если нужно

4. Размер приложения (~100 МБ) нормальный
   Включает Python и все библиотеки
