  Image Data Annotator v1.0.0 - Инструкция по установке
═══════════════════════════════════════════════════════════════

📥 УСТАНОВКА

1. Скачайте установщик:
   ImageDataAnnotator-Setup-v1.0.0.exe

2. Запустите установщик двойным кликом

3. Следуйте инструкциям мастера установки:
   ✓ Выберите папку установки (по умолчанию: C:\Program Files\Image Data Annotator)
   ✓ Создайте ярлык на рабочем столе (рекомендуется)
   ✓ Нажмите "Установить"

4. Дождитесь завершения установки

5. Запустите приложение:
   - Из меню Пуск: Image Data Annotator
   - С рабочего стола (если создали ярлык)

═══════════════════════════════════════════════════════════════
🚀 БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════

1. ПОДГОТОВЬТЕ ДАННЫЕ:

   ✓ Папка с изображениями:
     - Файлы: 1-1.png, 1-2.png, 2-1.png и т.д.
     - Формат номера: позиция-номер.png

   ✓ Excel файл с данными:
     - Столбец A: № позиции (1, 2, 3...)
     - Столбец S: Наименование поставщика
     - Столбец U: КПП
     - Столбец V: ИНН

   ✓ Файл печати (опционально):
     - Формат: PNG с прозрачным фоном

2. НАСТРОЙТЕ ПРИЛОЖЕНИЕ:

   Вкладка "Файлы":
   ✓ Папка с исходными изображениями → Укажите папку A
   ✓ Папка для результатов → Укажите папку B
   ✓ Файл Excel → Выберите ваш Excel файл
   ✓ Печать/подпись → Выберите файл печати (если нужно)

   Вкладка "Данные":
   ✓ Включите нужные поля (ИНН, КПП, Поставщик)
   ✓ Проверьте номера столбцов
   ✓ Настройте фиксированные тексты

   Вкладка "Вывод":
   ✓ Выберите формат (PDF рекомендуется)
   ✓ Выберите расположение панели (Слева или Снизу)

3. НАСТРОЙТЕ ПОЗИЦИИ (если нужно):

   Вкладка "Предпросмотр":
   ✓ Нажмите "Обновить предпросмотр"
   ✓ Включите "🎯 Режим редактирования"
   ✓ Перетащите надписи и печать мышкой
   ✓ Смещения сохранятся автоматически

4. ЗАПУСТИТЕ ОБРАБОТКУ:

   ✓ Нажмите "ЗАПУСТИТЬ ОБРАБОТКУ"
   ✓ Дождитесь завершения (прогресс показывается внизу)
   ✓ Найдите результаты в папке B

═══════════════════════════════════════════════════════════════
❓ ЧАСТЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════

Q: Приложение не запускается / Windows Defender блокирует
A: При первом запуске Windows может показать предупреждение.
   Нажмите "Подробнее" → "Выполнить в любом случае"

Q: Не находит изображения
A: Проверьте формат имен файлов: должно быть "номер-номер.png"
   Например: 1-1.png, 1-2.png, 2-1.png

Q: Нет данных для некоторых файлов
A: Убедитесь, что в Excel есть записи для этих позиций
   Проверьте столбец A (№ п.п.)

Q: Текст не появляется на изображениях
A: 1. Проверьте, что поля включены на вкладке "Данные"
   2. Обновите предпросмотр перед обработкой
   3. Проверьте цвет текста (должен отличаться от фона)

Q: Печать не отображается
A: 1. Убедитесь, что включено "Добавить печать/подпись"
   2. Используйте PNG с прозрачным фоном
   3. Проверьте размер файла печати

Если возникли проблемы:
1. Проверьте журнал обработки в приложении (вкладка "Предпросмотр")
2. Убедитесь, что формат файлов соответствует требованиям
3. Попробуйте обработать 1-2 файла для теста
